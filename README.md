# extractfiles
**导出项目中的增量文件**

# 环境 #
- jdk1.8
- maven 3.3.9

# 项目说明 
- **原文件路径为：**
source_path后的路径+输入路径的每一行
- **复制出来的路径为：**
des_path后的路径+输入路径的每一行
